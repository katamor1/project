<!-- Defines the cross-cutting context hygiene rule for long-running Bob workflows. -->
<!-- This exists so Bob conversations stay focused and do not mix planning, evidence gathering, and execution. -->
<!-- RELEVANT FILES: ibm-bob/mode-pack/pilot-runbook.md, ibm-bob/mode-pack/legacy-search-guide.md, ibm-bob/mode-pack/sdlc-guide.md -->
長い作業は新しい会話に切ってください。

plan と execution を同じ会話に混ぜないでください。

legacy-search と SDLC authoring も同じ会話で混ぜすぎないでください。

大きい仕事では、今どの lane かを最初に短く明示してください。

古い前提が残ると危険な時は、新しい会話でやり直してください。
