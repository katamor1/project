# eval_only_scope

この環境では、JSONテンプレート式の評価対象を仕様書生成とレビューに限定する。

## Allowed

- 基本設計書のレビュー
- 詳細設計書JSONの作成
- 詳細設計書DOCXの生成
- 詳細設計書レビュー
- 機能テスト仕様書JSONの作成
- 機能テスト仕様書DOCXの生成
- 機能テスト仕様書レビュー

## Forbidden

- ソースコード変更
- 単体テストコード作成
- ビルド実行
- テスト実行
- 実機接続

## Stage Gate

各ステージは `JSON -> DOCX -> validate` の順で完了とする。
レビューで `block` となった場合は、次ステージを自動継続せず、人間チェックポイントを要求する。
